Taretto

Targeted Testing Made Simple

What is Taretto?

Taretto is a testing framework written in Python which helps you to build your end-to-end tests.

  • Class-based UI modeling, abstracted from Selenium
  • Step-based navigation between pages and sections
  • Multi-implementation/endpoint handling, simultaneously support REST, DB, UI, Command line, all from the same framework
  • Start with an Appliance

Why Taretto?

Abstract your web pages into widgets and views

Interact with objects instead of Selenium and XPath

Automagically navigate to a page by just specifying the destination

Transparently use a REST API or user interface via the same method
